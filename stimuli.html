Set Up Your Stimuli

Overview of Stimuli Pool

This is where you put the same type of stimuli in. You can click "Add Item" to populate each of your stimuli pool. You can also click the little "+" on the top of the stimuli pool box to add more stimuli pools.
Q: How to decide which pool I should put the stimulus into?
A: For example, in IAT, we have negative words and positive words, which stand for two categories. Therefore, they should be put into two stimuli pools.

Type / Image URL or Text Context / Count

This is where you need to select the type of your stimuli, whether it is an image or text. If it is text, just copy and paste the text stimuli. If it is an image, load your image to an online library and paste its URL here. Put in the counts for each stimulus, i.e., how many times you want the stimulus to appear in your study.
Q: How to decide the amount of counts for stimuli?
A: For example, in IAT, we want the same word to repeat twice during one round. We just put "2" for its count.

Shuffle / Restrict Max Repeat

Here, the SP-Builder has a function called “Shuffle”that helps you randomize the order of all the items in your stimuli. Once you activate the “Shuffle” function, the SP-Builder will embed the JavaScript in your survey to generate a unique randomized order of items for each stimulus for each participant. Each stimulus has its own “Shuffle” function and needs to be activated separately.
Restrict max repeat In some cases, researchers want to prevent the same item from being repeated multiple times in succession, so they need to activate the “restrict max repeat” function and indicate the number of times that they allow the item to be repeatedly presented. For example, if you put “2,” then the aforementioned stimuli cannot appear more than twice in succession.
Q: What if I don't click shuffle?
A: Our website will play the stimuli from the first one until the last one and repeat the same sequence.
