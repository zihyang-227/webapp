Prime Your Stimuli

Overview of Priming

The “Priming” function is a powerful function of the SP-Builder designated for researchers who need to set up a selected item (e.g., target or control item). For example, as a common practice in psychology research, researchers in many sequential priming paradigms need a target item to compare with a control stimulus for their research purpose.


Set Up Your Target/Control Stimuli

You need to first click Add Priming Stimulus.

Give the name for your target stimuli in "Name", and add the stimuli pool to choose from in "Included items".

Then click Add Priming Stimulus again to set your control stimuli, where you can exclude target stimuli in "Excluded items".


Override Stimuli Count

In some studies, researchers want to present the selected item (e.g., Target_1) more frequently than the other items. They can use the “Override stimuli count” function to adjust the number of counts they wish for their selected item.
